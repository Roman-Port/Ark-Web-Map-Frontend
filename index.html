FAQ
How does Delta Web Map work?
Delta Web Map is installed as an ARK Workshop mod. Once installed, it'll start syncing all tribe dinos, structures, and items to our servers. Then, players log in with Steam and are shown only data from their own tribe.
How do I view a server I play on with Delta Web Map installed?
When you join a server with Delta Web Map installed, it'll appear in the Delta Web Map server list within a minute of you joining. You don't have to do anything special.
How does Delta Web Map know who is an admin?
Players who log in as an admin on the game server are automatically granted admin access on Delta Web Map.
How much does Delta Web Map cost?
It's free to both server owners and players! We'll likely add a paid tier with more features in the future, but the core will always be free.
Can I restrict what players can access?
Yep! You can disable many features of Delta Web Map as an admin. You're also able to temporarily lock players out of Delta Web Map.
Will in-game players notice Delta Web Map on my server?
Delta Web Map only quietly runs in the background on the server. Due to the nature of ARK mods, players need to download the mod to join, but it won't do anything.
Are there any hosting requirements for Delta Web Map?
Nope, except that you must be playing on Steam. Make sure your server has access to outgoing traffic to sync-nossl-prod.deltamap.net on ports 80 and 443.
Does Delta Web Map support consoles?
Nope, sorry. Consoles don't have mod support, which is required to install the Delta Web Map Workshop mod.
Can I use other mods?
Yes! All mods will work, but some content added by mods won't be easily readable in Delta Web Map unless the mod has been specifically supported by us.
Can I use modded maps?
Sort of. You won't be able to see map tiles, but you can use all other features of Delta Web Map.
