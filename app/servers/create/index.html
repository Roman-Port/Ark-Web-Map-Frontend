The Ark Web Map Companion Mod is used to push realtime events, show online players, send notifications, and more.

To install, download the mod from the Steam Workshop and open the "GameUserSettings.ini" file in "ShooterGame/Saved/Config/WindowsServer". Add the "ActiveMods=1775128245", or add the mod ID 1775128245.
